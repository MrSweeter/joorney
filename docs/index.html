How to Install a Chrome Extension with Developer Mode?

Follow these steps to download, install and configure a Chrome extension with developer mode:

Download the extension

  1. Open the GitHub repository that contains the extension you want to install. The GitHub repository for this extension is here.
  2. Download the project. If you are not comfortable using git commands in a terminal, you can follow the instructions in "Download" tab. Else you can check the "GIT" tab.
    1. You can click on the Code button in the top right corner of the repository page, and then select Download ZIP. This will download the ZIP file to your computer.
    2. Once the download is done, unzip the downloaded file to extract the extension files and folders. You can use any software that can unzip files, such as WinRAR or 7-Zip. You should see a folder that contains files/directories such as manifest.json, background, content, popup, etc. These are the files that make up the extension.
    You can git clone the project and git switch/checkout on the desired branch.

Install the extension

  1. Open Chrome and go to chrome://extensions. This will open the Extensions page, where you can see all the extensions that are installed on your browser.
  2. Enable developer mode by toggling the switch in the top right corner of the Extensions page. You should see some additional options appear below the switch.
  3. Click on Load unpacked and select the folder that contains the extension files and folders. This will load the extension into your browser and add it to the list of extensions on the Extensions page.
  4. You can now use the extension as normal. You can also manage its settings, permissions, or remove it from your browser by clicking on the Details button next to its name on the Extensions page.

Configure the extension

  1. Open extension details page by clicking on the Details button next to its name on the Extensions page.
  2. In the details, find the line Extension options an click on it. This will open the extension option page into your browser
  3. To configure a quicker access to this page and quick extension feature access, you can pin it in the browser navigation bar
    • Open extension popup by clicking on the in the top right corner of the browser window.
    • Find this extension name Joorney and click on the . You should see the extension icon appearing to the left of the
  4. Once the extension is pinned in your browser navigation bar, you have two options:
    • Left click: you will see a new user interface that lets you turn on or off the extension feature with ease.
    • Right click: you will see a contextual menu that let you access to Options or to Manage extension (the details page)
  5. You have learned the basic steps to access the extension options. Now, you can explore them as you wish.
Once setup, don't hesitate to export your configuration via the option page, check top right corner of it
