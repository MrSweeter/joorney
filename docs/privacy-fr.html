Politique de confidentialité

Dernière mise à jour : 3 juillet 2026

Cette page décrit les données que l'extension Chrome Joorney peut utiliser ou stocker pour fonctionner correctement.

Cette politique s'applique à l'extension Joorney et aux fonctionnalités fournies dans le navigateur.

Cette politique de confidentialité est accessible publiquement et est liée dans le champ de confidentialité du tableau de bord du Chrome Web Store.

English Version

1. Données collectées

Joorney ne collecte pas de données personnelles pour de l'analyse ou du tracking. L'extension utilise les informations uniquement pour fournir ses fonctionnalités et améliorer votre expérience Odoo.

1.1 Données de navigation et URL

Joorney peut lire l'URL de l'onglet actif et les pages Web ouvertes dans le navigateur afin de détecter si vous êtes sur un site Odoo, d'activer les fonctionnalités pertinentes décrites dans la fiche du Chrome Web Store, et de proposer des actions contextuelles. Cette activité de navigation est collectée uniquement pour les fonctionnalités visibles et dédiées à Odoo.

1.2 Données de session Odoo

Lorsque vous utilisez Joorney sur un site Odoo, l'extension peut lire des informations présentes dans la page Odoo, comme l'objet de session interne (odoo.__session_info__), pour déterminer votre état de session et proposer des actions utiles. Ces données ne sont utilisées que localement dans votre navigateur pour faire fonctionner l'extension.

1.3 Cookies

Joorney utilise l'accès aux cookies uniquement pour la fonctionnalité de changement de thème Odoo. Elle lit et écrit les cookies suivants sur le domaine Odoo :

  • color_scheme
  • configured_color_scheme

1.4 Données de stockage local et synchronisé

L'extension stocke localement vos préférences, réglages de fonctionnalités et données de cache dans le stockage chrome.storage.local et chrome.storage.sync. Ces données comprennent par exemple :

  • Préférences d'activation des fonctionnalités
  • Origines désactivées
  • Données de cache pour accélérer les appels internes
  • Paramètres de texte ou d'interface utilisateur

1.5 Données de localisation

Joorney peut accéder à votre position géographique uniquement si vous cliquez explicitement sur le bouton d'obtention de la position dans les paramètres du thème de l'extension. Cela sert uniquement à calculer les heures de lever/coucher du soleil pour le mode sombre automatique.

1.6 Requêtes externes

Joorney effectue des appels réseau vers les services suivants :

  • GitHub pour récupérer les données publiques d'annonce et de versions.
  • SunriseSunset.io pour calculer les horaires de lever et coucher du soleil lorsque l'option correspondante est activée.

Lorsque le mode sombre automatique avec localisation est activé, Joorney peut envoyer vos coordonnées de latitude et de longitude à SunriseSunset.io afin de calculer les heures de lever et coucher du soleil. Ces données de localisation sont utilisées uniquement pour cette fonctionnalité et ne sont pas conservées à d'autres fins.

Les autres requêtes de service n'impliquent pas de données personnelles identifiables. Joorney ne partage pas les données personnelles des utilisateurs avec des tiers pour la publicité, l'analyse ou le profilage.

2. Utilisation des données

Les données sont utilisées uniquement pour :

  • Déterminer si vous naviguez sur un site Odoo
  • Activer ou désactiver les fonctionnalités adaptées au site
  • Accéder aux API Odoo en même temps que vous, pour fournir des actions comme l'ouverture de menus ou la lecture de données
  • Garder vos paramètres et préférences d'extension
  • Calculer des horaires de thème si vous activez l'option de localisation

Toute utilisation des données utilisateur est limitée aux fonctionnalités décrites ici et dans la fiche du Chrome Web Store. Joorney n'utilise pas les données utilisateur pour la publicité, le profilage ou les décisions de crédit.

Joorney ne vend pas, ne loue pas et ne partage pas vos données personnelles à des tiers.

3. Retention et suppression

Les paramètres et données de cache restent stockés tant que l'extension est installée. Vous pouvez supprimer ces données en effaçant le stockage de l'extension depuis les options Chrome ou en désinstallant Joorney.

4. Absence de suivi externe

Joorney n'utilise pas de services d'analyse ou de suivi tiers (comme Google Analytics, Sentry, Amplitude, Mixpanel, Hotjar, etc.).

5. Permissions Chrome requises

Le manifeste de l'extension déclare les permissions suivantes. Ces permissions sont utilisées uniquement pour les fonctionnalités décrites dans cette politique et dans la fiche du Chrome Web Store.

  • tabs : pour lire l'URL de l'onglet actif et répondre aux événements d'onglet.
  • webNavigation : pour détecter les changements de navigation et charger dynamiquement les fonctionnalités.
  • cookies : pour lire et écrire les cookies Odoo liés au thème.
  • storage : pour conserver vos préférences et données de cache.
  • contextMenus : pour créer des menus contextuels dans le navigateur.
  • webRequest : pour détecter l'état des requêtes HTTP utilisées par certaines fonctionnalités.
